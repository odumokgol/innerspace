# Innerspace - TCI 심리검사 안내 페이지

링크인바이오 스타일의 TCI 기질·성격 검사 안내 모바일 웹페이지

## 🌐 라이브 URL

https://innerspace-puce.vercel.app

## 📁 저장소

https://github.com/odumokgol/innerspace

## 🛠 기술 스택

- 순수 HTML / CSS / JavaScript (빌드 불필요)
- Vercel 무료 플랜으로 배포

## 📄 파일 구조

```
tci-guide/
├── index.html    ← 링크인바이오 메인 페이지
├── style.css     ← 프리미엄 다크 테마 스타일
├── script.js     ← 토글·리플 인터랙션
└── .gitignore
```

## 📱 페이지 구성 (위→아래)

1. 프로필 헤더 (아바타 + 소개)
2. 🆓 무료 검사 신청 — 커리어넷, 워크넷
3. 🏛️ 공식 기관 — KPTI, 국립특수교육원
4. 📖 TCI 알아보기 — 아코디언 정보 패널 3개
5. 🏥 상담센터 찾기 — 네이버·카카오맵

## 🚀 수정 후 배포 방법

```powershell
cd "d:\Project_X\tci-guide"
git add .
git commit -m "수정 내용 메모"
git push
```

> `git push` 후 Vercel이 자동으로 재배포됩니다.
